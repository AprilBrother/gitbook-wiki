Important Note: The AT command must be end with "\\r\\n". For Arduino,
must type the AT command like this

`Serial1.print("AT+GMR\r\n");`

Not like this:

`Serial1.println("AT+GMR");`

So the hardwareSerialMonitor sketch will not work
    now.
